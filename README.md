# QuizMaster-Dynamic-Full-Stack-Application

Quiz Master App is a responsive, API‑driven quiz platform enabling real‑time question rendering and score tracking via lightweight JSON. It integrates a synchronized countdown timer and instant feedback loops with async JavaScript, delivering smooth, mobile‑first interaction across browsers.

---

## ✨ Features

- 🔐 **Login & Authentication** – Secure entry point with polished UI.
- ⚡ **API‑Driven Quiz Engine** – Real‑time question rendering and score tracking via lightweight JSON storage.
- ⏱️ **Countdown Timer** – Synchronized per‑question timer with instant feedback loops using async JavaScript.
- 📊 **Score History** – Smooth scrolling leaderboard with styled scrollbar and animated close button.
- 📱 **Responsive Design** – Mobile‑first layout ensuring cross‑browser compatibility.
- 🎨 **UI/UX Enhancements** – Gradients, hover animations, micro‑interactions, and accessibility‑friendly layouts.

---

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Backend:** Node.js + Express
- **Storage:** Lightweight JSON for questions & history
- **Deployment:** GitHub Pages, Netlify, or Render

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Rushikesh-tetwar/QuizMaster-Dynamic-Full-Stack-Application
   ```
2. Navigate into the project:
   ```bash
   cd QuizMaster-Dynamic-Full-Stack-Application
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Run locally:
   - **Development mode (auto‑reload with Nodemon):**
   ```bash
   npm run dev
   ```
   - **Production mode:**
   ```bash
   npm start
   ```
